Job Portal (Flask + MySQL)

Overview

This is a full-stack Job Portal web application built using Flask, MySQL, HTML, and CSS.
It allows users to register, login, and post job listings dynamically.

---

Features 

* User Registration & Login
* Session-based Authentication
* Job Posting System
* View All Jobs
* Logout Functionality
* Protected Dashboard (only logged-in users)

Tech Stack

* Backend:Flask (Python)
* Frontend:HTML, CSS
* Database: MySQL
* Version Control:Git & GitHub

---

Project Structure

job_portal/
│── app.py
│── templates/
│── static/
│── requirements.txt

 Usage

1. Register a new user
2. Login with credentials
3. Access dashboard
4. Post jobs and view listings
